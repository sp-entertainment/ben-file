# Ben File

Ben File is a Visual Studio Code extension that adds support for the **ben** file type and related workflows.

## Features

- **Hello World** — Run the command **Ben File: Hello World** from the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) to verify the extension is loaded.

## Requirements

- Visual Studio Code **1.110** or newer.

## Extension Settings

This version does not add workspace settings yet.
